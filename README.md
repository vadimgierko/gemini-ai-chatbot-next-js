# Gemini AI Chatbot

## Features

<ul>
    <li>Sign in using Google Account to use the app for free*.
    <br />
        <small>
            *This is needed to prevent abusing app API routes by malicious users.
        </small>
    </li>
    <li>Text based temporary (not stored in database) chat with streaming response with Gemini.</li>
    <li>Set system instruction (context).</li>
    <li>Markdown support.</li>
    <li>Code highlighting.</li>
    <li>Dark mode</li>
</ul>

## Tech Stack
    
<ul>
    <li>Next.js 15+</li>
    <li>React 19+</li>
    <li>Gemini 2.0 Flash</li>
    <li>Firebase 11+</li>
    <li>TypeScript 5+</li>
    <li>Bootstrap 5.3+</li>
    <li>Markdown</li>
</ul>
